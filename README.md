This is a blog web app built with Node.js, Express, and EJS. 
It lets you add and delete blog posts, stored in pg database. 

# DEPLOYED @ https://postline.onrender.com

Features
  Add blog posts
  Delete blog posts
  database now

To Run
  Clone Project
  Run npm install
  Run node index.js
